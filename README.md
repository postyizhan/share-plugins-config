# 📦 安装

安装 EssentialsX

覆盖文件后重启服务端

# ⚠️ 注意

我不会去支持 EssentialsX 之外的提供tpa的插件

# ⚙️ 配置

在其他菜单中打开此菜单

指 Invero 的其他菜单，不是指其他菜单插件
```
    material: stone
      action:
       context set ignoreOpenEvents to true
       menu open 此菜单名称
```
